JavaScript Introduction

JavaScript ஐ அறிமுகப்படுத்துதல்

JavaScript என்ன செய்ய முடியும்?

JavaScript என்பது வலைத்தளத்தின் நிரலாக்க மொழியாகும்.

இது தரவுகளை கணக்கிடலாம், கையாளலாம் மற்றும் சரிபார்க்கலாம்.

இது HTML மற்றும் CSS இரண்டையும் புதுப்பிக்கலாம் மற்றும் மாற்றலாம்.

JavaScript HTML உள்ளடக்கத்தை மாற்ற முடியும்

பல JavaScript HTML முறைகளில் ஒன்று getElementById() ஆகும்.

கீழே உள்ள எடுத்துக்காட்டு ஒரு HTML உறுப்பை (id="demo" உடன்) "கண்டுபிடிக்கிறது", மற்றும் உறுப்பு உள்ளடக்கத்தை (innerHTML) "Hello JavaScript" என மாற்றுகிறது:

எடுத்துக்காட்டு

document.getElementById("demo").innerHTML = "Hello JavaScript";

JavaScript இரட்டை மற்றும் ஒற்றை மேற்கோள்கள் இரண்டையும் ஏற்கிறது:

document.getElementById('demo').innerHTML = 'Hello JavaScript';

நேரடி டெமோ:

இந்த உரை JavaScript மூலம் மாறும்

JavaScript HTML பண்பு மதிப்புகளை மாற்ற முடியும்

இந்த எடுத்துக்காட்டில் JavaScript ஒரு <img> குறிச்சொல்லின் src (மூல) பண்பின் மதிப்பை மாற்றுகிறது:

💡

JavaScript HTML பாணிகளை (CSS) மாற்ற முடியும்

ஒரு HTML உறுப்பின் பாணியை மாற்றுவது, ஒரு HTML பண்பை மாற்றுவதன் மாறுபாடு ஆகும்:

எடுத்துக்காட்டு

document.getElementById("demo").style.fontSize = "35px";

நேரடி டெமோ:

இந்த உரையின் அளவு JavaScript மூலம் மாறும்

JavaScript HTML உறுப்புகளை மறைக்க முடியும்

HTML உறுப்புகளை மறைப்பது, display பாணியை மாற்றுவதன் மூலம் செய்யப்படலாம்:

எடுத்துக்காட்டு

document.getElementById("demo").style.display = "none";

நேரடி டெமோ:

இந்த உறுப்பு JavaScript மூலம் மறைக்கப்படும்/காட்டப்படும்

JavaScript HTML உறுப்புகளை காட்ட முடியும்

மறைக்கப்பட்ட HTML உறுப்புகளை காட்டுவதும் display பாணியை மாற்றுவதன் மூலம் செய்யப்படலாம்:

எடுத்துக்காட்டு

document.getElementById("demo").style.display = "block";

உங்களுக்குத் தெரியுமா?

JavaScript மற்றும் Java ஆகியவை இரண்டு முற்றிலும் வித்தியாசமான மொழிகளாகும், கருத்து மற்றும் வடிவமைப்பு இரண்டிலும்.

JavaScript 1995 ஆம் ஆண்டில் Brendan Eich என்பவரால் கண்டுபிடிக்கப்பட்டது, மற்றும் 1997 ஆம் ஆண்டில் ECMA தரநிலையாக மாறியது.

ECMA-262 என்பது தரநிலையின் அதிகாரப்பூர்வ பெயர். ECMAScript என்பது மொழியின் அதிகாரப்பூர்வ பெயர்.

JavaScript பயிற்சிகள்

இந்த டுடோரியலில் உள்ள பல அத்தியாயங்கள் உங்கள் அறிவு நிலையைச் சரிபார்க்கக்கூடிய பயிற்சியுடன் முடிகின்றன.

❓ உண்மை அல்லது பொய்.

JAVA என்பது JavaScript க்கான சுருக்கமா?

True
✗ தவறு! JAVA என்பது JavaScript க்கான சுருக்கம் அல்ல. அவை இரண்டு வெவ்வேறு நிரலாக்க மொழிகள்
False
✓ சரி! JavaScript மற்றும் Java ஆகியவை இரண்டு தனித்துவமான நிரலாக்க மொழிகள். JavaScript ஐ முதலில் "LiveScript" என்று அழைக்கப்பட்டது, பின்னர் சந்தைப்படுத்தல் காரணங்களுக்காக JavaScript என மறுபெயரிடப்பட்டது
