JavaScript டுடோரியல்

JavaScript கற்றுக்கொள்வது எளிது.

இந்த டுடோரியல் அடிப்படை JavaScript முதல் 2025 பதிப்பு வரை அனைத்தையும் உள்ளடக்கியது.

ஒவ்வொரு அத்தியாயத்திலும் எடுத்துக்காட்டுகள்

எங்கள் "Try it Yourself" தொகுப்பியுடன், நீங்கள் மூலக் குறியீட்டைத் திருத்தலாம் மற்றும் முடிவைக் காணலாம்.

எடுத்துக்காட்டு

எனது முதல் JavaScript

<button onclick="displayDate()">Click me to display Date and Time</button>
<div id="dateOutput"></div>

<script>
function displayDate() {
    document.getElementById("dateOutput").innerHTML = Date();
}
</script>

இது எவ்வாறு செயல்படுகிறது என்பதைப் பார்க்கவும்:

மெனுவைப் பயன்படுத்தவும்

இந்த டுடோரியலை மெனுவில் பட்டியலிடப்பட்டுள்ள வரிசையில் படிக்க பரிந்துரைக்கிறோம்.

உங்களிடம் பெரிய திரை இருந்தால், மெனு எப்போதும் இடதுபுறத்தில் இருக்கும்.

உங்களிடம் சிறிய திரை இருந்தால், மேல் மெனு அடையாளம் ☰ என்பதைக் கிளிக் செய்வதன் மூலம் மெனுவைத் திறக்கவும்.

எடுத்துக்காட்டுகள் மூலம் கற்றுக்கொள்ளுங்கள்

எடுத்துக்காட்டுகள் 1000 வார்த்தைகளை விட சிறந்தவை.

எடுத்துக்காட்டுகள் பெரும்பாலும் உரை விளக்கங்களை விட புரிந்துகொள்வது எளிது.

இந்த டுடோரியல் அனைத்து விளக்கங்களையும் தெளிவுபடுத்தும் "Try it Yourself" எடுத்துக்காட்டுகளுடன் நிரப்புகிறது.

📝 குறிப்பு:

நீங்கள் அனைத்து எடுத்துக்காட்டுகளையும் முயற்சித்தால், JavaScript பற்றி மிகக் குறுகிய காலத்தில் நிறைய கற்றுக்கொள்வீர்கள்!

ஏன் JavaScript கற்க வேண்டும்?

JavaScript என்பது அனைத்து வலை மேம்பாட்டாளர்களும் கற்றுக்கொள்ள வேண்டிய 3 மொழிகளில் ஒன்றாகும்:

  • வலைப்பக்கங்களின் உள்ளடக்கத்தை வரையறுக்க HTML
  • வலைப்பக்கங்களின் வடிவமைப்பைக் குறிப்பிட CSS
  • வலைப்பக்கங்களின் நடத்தையை நிரலாக்க JavaScript

கற்றல் வேகம்

இந்த டுடோரியலில், கற்றல் வேகம் உங்கள் தேர்வு.

எல்லாம் உங்களுடையது.

நீங்கள் போராடுகிறீர்கள் என்றால், ஓய்வு எடுத்துக் கொள்ளுங்கள் அல்லது பொருளை மீண்டும் படிக்கவும்.

"Try-it-Yourself" எடுத்துக்காட்டுகள் அனைத்தையும் நீங்கள் புரிந்துகொள்கிறீர்களா என்பதை எப்போதும் உறுதிப்படுத்திக் கொள்ளுங்கள்.

புத்திசாலியான நிரலாளராக மாற ஒரே வழி:

குறியிடுவதைப் பயிற்சி செய்யுங்கள்! குறியிடுவதைப் பயிற்சி செய்யுங்கள்! குறியிடுவதைப் பயிற்சி செய்யுங்கள்!

பொதுவாக கேட்கப்படும் கேள்விகள்

JavaScript எப்படி பெறுவது?

JavaScript எங்கே பதிவிறக்கம் செய்யலாம்?

JavaScript இலவசமா?

நீங்கள் JavaScript பெறவோ அல்லது பதிவிறக்கம் செய்யவோ தேவையில்லை.

JavaScript ஏற்கனவே உங்கள் கணினி, உங்கள் டேப்லெட் மற்றும் உங்கள் ஸ்மார்ட்போனில் உங்கள் உலாவியில் இயங்குகிறது.

JavaScript அனைவருக்கும் இலவசமாகப் பயன்படுத்தப்படலாம்.

JavaScript பயிற்சிகள்

இந்த டுடோரியலில் உள்ள பல அத்தியாயங்கள் உங்கள் அறிவு நிலையைச் சரிபார்க்கக்கூடிய பயிற்சியுடன் முடிகின்றன.

❓ மாறிக்கு மதிப்பை ஒதுக்க சரியான தொடரியல் எது?

x : 5
✗ தவறு! இந்த தொடரியல் JavaScript இல் பயன்படுத்தப்படுவதில்லை
x = 5
✓ சரி! JavaScript இல் ஒதுக்கீட்டு ஆபரேட்டர் '=' ஆகும்
x == 5
✗ தவறு! '==' என்பது சமன்பாட்டு ஆபரேட்டர், ஒதுக்கீடு அல்ல
x -> 5
✗ தவறு! '->' ஆபரேட்டர் JavaScript இல் இல்லை
