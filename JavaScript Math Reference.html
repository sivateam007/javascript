முழுமையான Math குறிப்பு

📅 புதுப்பிக்கப்பட்ட தேதி:

ஜூலை 2025 இல் புதுப்பிக்கப்பட்டது

JavaScript Math பொருளின் அனைத்து பண்புகள் மற்றும் முறைகளின் முழுமையான குறிப்பு:

பெயர் விளக்கம்
abs(x) x இன் முழுமையான மதிப்பை வழங்குகிறது
acos(x) x இன் arccosine ஐ ரேடியன்களில் வழங்குகிறது
acosh(x) x இன் ஹைபர்போலிக் arccosine ஐ வழங்குகிறது
asin(x) x இன் arcsine ஐ ரேடியன்களில் வழங்குகிறது
asinh(x) x இன் ஹைபர்போலிக் arcsine ஐ வழங்குகிறது
atan(x) -PI/2 மற்றும் PI/2 ரேடியன்களுக்கு இடையே ஒரு எண் மதிப்பாக x இன் arctangent ஐ வழங்குகிறது
atan2(y, x) அதன் வாதங்களின் பங்கீட்டின் arctangent ஐ வழங்குகிறது
atanh(x) x இன் ஹைபர்போலிக் arctangent ஐ வழங்குகிறது
cbrt(x) x இன் கன மூலத்தை வழங்குகிறது
ceil(x) அருகிலுள்ள முழு எண்ணுக்கு மேலே x ஐ மாற்றுகிறது
clz32(x) x இன் 32-பிட் பைனரி பிரதிநிதித்துவத்தில் முன்னணி பூஜ்ஜியங்களின் எண்ணிக்கையை வழங்குகிறது
cos(x) x இன் cosine ஐ வழங்குகிறது (x ரேடியன்களில் உள்ளது)
cosh(x) x இன் ஹைபர்போலிக் cosine ஐ வழங்குகிறது
E Euler's எண்ணை வழங்குகிறது (தோராயமாக 2.718)
exp(x) Ex இன் மதிப்பை வழங்குகிறது
expm1(x) Ex கழித்தல் 1 இன் மதிப்பை வழங்குகிறது
f16round(x) அருகிலுள்ள முழு எண்ணுக்கு கீழே x ஐ மாற்றுகிறது
floor(x) அருகிலுள்ள முழு எண்ணுக்கு கீழே x ஐ மாற்றுகிறது
fround(x) ஒரு எண்ணின் அருகிலுள்ள (32-பிட் ஒற்றை துல்லியம்) மிதவை பிரதிநிதித்துவத்தை வழங்குகிறது
LN2 2 இன் இயற்கை மடக்கையை வழங்குகிறது (தோராயமாக 0.693)
LN10 10 இன் இயற்கை மடக்கையை வழங்குகிறது (தோராயமாக 2.302)
log(x) x இன் இயற்கை மடக்கையை வழங்குகிறது
log10(x) x இன் அடிப்படை-10 மடக்கையை வழங்குகிறது
LOG10E E இன் அடிப்படை-10 மடக்கையை வழங்குகிறது (தோராயமாக 0.434)
log1p(x) 1 + x இன் இயற்கை மடக்கையை வழங்குகிறது
log2(x) x இன் அடிப்படை-2 மடக்கையை வழங்குகிறது
LOG2E E இன் அடிப்படை-2 மடக்கையை வழங்குகிறது (தோராயமாக 1.442)
max(x1,x2,..) அதிக மதிப்பைக் கொண்ட எண்ணை வழங்குகிறது
min(x1,x2,..) குறைந்த மதிப்பைக் கொண்ட எண்ணை வழங்குகிறது
PI PI ஐ வழங்குகிறது (தோராயமாக 3.14)
pow(x, y) y இன் சக்திக்கு x இன் மதிப்பை வழங்குகிறது
random() 0 மற்றும் 1 க்கு இடையே ஒரு சீரற்ற எண்ணை வழங்குகிறது
round(x) அருகிலுள்ள முழு எண்ணுக்கு x ஐ மாற்றுகிறது
sign(x) ஒரு எண்ணின் அடையாளத்தை வழங்குகிறது (அது நேர்மறையா, எதிர்மறையா அல்லது பூஜ்யமா என்பதை சரிபார்க்கிறது)
sin(x) x இன் sine ஐ வழங்குகிறது (x ரேடியன்களில் உள்ளது)
sinh(x) x இன் ஹைபர்போலிக் sine ஐ வழங்குகிறது
sqrt(x) x இன் வர்க்கமூலத்தை வழங்குகிறது
SQRT1_2 1/2 இன் வர்க்கமூலத்தை வழங்குகிறது (தோராயமாக 0.707)
SQRT2 2 இன் வர்க்கமூலத்தை வழங்குகிறது (தோராயமாக 1.414)
tan(x) ஒரு கோணத்தின் tangent ஐ வழங்குகிறது
tanh(x) ஒரு எண்ணின் ஹைபர்போலிக் tangent ஐ வழங்குகிறது
trunc(x) ஒரு எண்ணின் (x) முழு எண் பகுதியை வழங்குகிறது

💡 பயன்பாட்டு உதவிக்குறிப்பு:

இந்த Math முறைகள் அனைத்தும் Math. முன்னொட்டுடன் பயன்படுத்தப்பட வேண்டும். எடுத்துக்காட்டாக: Math.abs(-5), Math.random(), Math.PI.

Math மாறிலிகளின் சுருக்கம்

Math.PI

π மதிப்பு (தோராயமாக 3.14159)

Math.PI  // 3.141592653589793

Math.E

Euler's எண் (தோராயமாக 2.71828)

Math.E   // 2.718281828459045

Math.SQRT2

2 இன் வர்க்கமூலம் (தோராயமாக 1.414)

Math.SQRT2  // 1.4142135623730951

Math.SQRT1_2

1/2 இன் வர்க்கமூலம் (தோராயமாக 0.707)

Math.SQRT1_2  // 0.7071067811865476

அதிகம் பயன்படுத்தப்படும் Math முறைகள்

Math.random()

0 (அடங்கும்) மற்றும் 1 (தவிர) இடையே சீரற்ற எண்ணை உருவாக்குகிறது

Math.random()  // 0.123456789...

Math.round()

அருகிலுள்ள முழு எண்ணுக்கு எண்ணை மாற்றுகிறது

Math.round(4.7)  // 5

Math.max() / Math.min()

வாதங்களில் அதிகபட்ச / குறைந்தபட்ச மதிப்பைக் கண்டறிகிறது

Math.max(1, 5, 3)  // 5
Math.min(1, 5, 3)  // 1

Math.sqrt()

ஒரு எண்ணின் வர்க்கமூலத்தைக் கணக்கிடுகிறது

Math.sqrt(64)  // 8

Math குறிப்பு பயிற்சி

❓ JavaScript இல் π (pi) மதிப்பைப் பெற எந்த Math பண்பு பயன்படுத்தப்படுகிறது?

Math.PIE
✗ தவறு! Math.PIE சரியான பண்பு அல்ல
Math.PI
✓ சரி! Math.PI π மதிப்பை வழங்குகிறது (தோராயமாக 3.14159)
Math.pi
✗ தவறு! JavaScript case-sensitive ஆகும். Math.pi வேலை செய்யாது, Math.PI மட்டுமே சரி
Math.getPi()
✗ தவறு! getPi() என்பது சட்டபூர்வமான JavaScript Math முறை அல்ல
