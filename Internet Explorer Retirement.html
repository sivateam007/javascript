Internet Explorer Retirement

உலாவி வரலாற்றில் ஒரு காலத்தின் முடிவைக் கற்றுக்கொள்ளுங்கள்

Internet Explorer ஓய்வு

Microsoft ஜூன் 15, 2022 அன்று Internet Explorer க்கான ஆதரவை முடித்துவிட்டது.

⚠️ முக்கிய அறிவிப்பு:

Internet Explorer 11 ஆனது ஜூன் 15, 2022 அன்று முற்றிலும் ஆதரவை இழந்துவிட்டது. இது வலை மேம்பாட்டில் ஒரு முக்கியமான கட்டத்தைக் குறிக்கிறது.

📅 தேதி வரிசை:

Google ஆதரவு

அக்டோபர் 1, 2021 முதல், Google Search இனி Internet Explorer 11 ஐ ஆதரிக்காது.

🔍 தேடல் பொறியில் தாக்கம்:

Google Search ஆதரவை நிறுத்தியது என்பது Internet Explorer பயனர்களுக்கு நவீன இணைய அனுபவத்தை அணுகுவது கடினமாக மாறியது என்பதைக் குறிக்கிறது.

Edge Legacy ஓய்வு

Microsoft ஜூன் 15, 2022 அன்று Edge Legacy க்கான ஆதரவை முடித்துவிட்டது.

Edge இன் முதல் பதிப்புகள் (12-18) "Edge Legacy" என்று குறிப்பிடப்படுகின்றன.

Edge என்பது Windows 10 க்கான இயல்புநிலை உலாவியாக இருந்தது. இது Microsoft இன் உலாவி எஞ்சின் EdgeHTML மற்றும் அவர்களின் Chakra JavaScript எஞ்சினால் கட்டப்பட்டது.

Edge Legacy க்கான Microsoft ஆதரவு மார்ச் 9, 2021 அன்று முடிவடைந்தது.

எதிர்கால Windows புதுப்பிப்புகள் Edge Legacy ஐ The New Edge உடன் மாற்றும்.

The New Edge

புதிய Edge 79 ஜனவரி 2020 இல் வெளியிடப்பட்டது மற்றும் Windows 7, 8, 10, மற்றும் 11 க்கு கிடைக்கிறது.

Chromium அடிப்படை: Chromium திறந்த-மூல எஞ்சினைப் பயன்படுத்துகிறது (Google Chrome உடன் ஒரே கோர்)
மேம்பட்ட பாதுகாப்பு: வேகமானது, மேலும் பாதுகாப்பானது, மற்றும் நவீன வலைத்தளங்களுடன் அதிக இணக்கத்தன்மை
நீட்டிப்புகள் ஆதரவு: Chrome நீட்டிப்புகளை ஆதரிக்கிறது
நவீன வாழ்க்கைச் சுழற்சி: Modern Lifecycle Policy ஐப் பின்பற்றுகிறது (தோராயமாக ஆறு வார சுழற்சியில் புதுப்பிப்புகள்)
பல தள ஆதரவு: macOS, iOS, மற்றும் Android க்கும் பதிவிறக்கம் செய்யலாம்
மாற்றீடு: Edge Legacy ஐ மாற்றுகிறது, இது மார்ச் 9, 2021 அன்று ஆதரவின் முடிவை எட்டியது

🔄 உலாவி எஞ்சின் மாற்றம்:

EdgeHTML இலிருந்து Chromium க்கு மாற்றம் என்பது Microsoft இன் உலாவி மூலோபாயத்தில் ஒரு குறிப்பிடத்தக்க மாற்றத்தைக் குறிக்கிறது, இது இணைய தரநிலைகள் மற்றும் நவீன வலைத் தொழில்நுட்பங்களுக்கான அதிக இணக்கத்தன்மையை நோக்கமாகக் கொண்டுள்ளது.

வலை மேம்பாட்டில் தாக்கம்

குறியீட்டு இணக்கத்தன்மை

Internet Explorer ஆதரவை நிறுத்துவது வலை மேம்பாட்டாளர்கள் IE க்கான வேலை செய்யும் குறியீடு மற்றும் ஹேக்குகளை நீக்க அனுமதிக்கிறது.

செயல்திறன்

IE பின்னோக்கு இணக்கத்தன்மையை கவனித்துக் கொள்ள வேண்டியதில்லை என்பதால் வலைத்தளங்கள் நவீன உலாவிகளுக்கு உகந்ததாக இருக்கும்.

பாதுகாப்பு

பழைய உலாவிகளை ஆதரிப்பதன் மூலம் வரும் பாதுகாப்பு பிரச்சினைகளைக் கையாள வேண்டிய அவசியமில்லை.

வள மேலாண்மை

குறைவான உலாவிகளை சோதிப்பதன் மூலம் மேம்பாட்டு நேரம் மற்றும் வளங்கள் சேமிக்கப்படுகின்றன.

மாற்றம் பரிந்துரைகள்

Edge உலாவிக்கு மாறவும்: பயனர்களை Microsoft Edge (Chromium) க்கு மாற்ற ஊக்குவிக்கவும்
நவீன JavaScript பயன்படுத்தவும்: ES6+ அம்சங்களை பயன்படுத்தி நவீன JavaScript குறியீட்டை எழுதவும்
CSS கட்டுப்பாடுகளை அகற்றவும்: IE க்கான CSS வேலை செய்யும் குறியீடு மற்றும் ஹேக்குகளை நீக்கவும்
நவீன API களைப் பயன்படுத்தவும்: Fetch API, Promises, async/await, மற்றும் பிற நவீன வலை API களைப் பயன்படுத்தவும்
சோதனை உத்திகளைப் புதுப்பிக்கவும்: உங்கள் சோதனை மேலாண்மையிலிருந்து IE ஐ நீக்கவும்

🚀 வலைத் தொழில்நுட்பங்களின் முன்னேற்றம்:

Internet Explorer ஓய்வு நவீன வலைத் தொழில்நுட்பங்களின் விரைவான தத்தெடுப்புக்கு வழிவகுத்தது, இது அனைத்து பயனர்களுக்கும் சிறந்த செயல்திறன், பாதுகாப்பு மற்றும் பயனர் அனுபவத்தை வழங்குகிறது.
