JavaScript இல் உள்ள அனைத்து வரிசை முறைகள் மற்றும் பண்புகளின் முழுமையான பட்டியல்:

பெயர் விளக்கம்
[ ] புதிய வரிசையை உருவாக்குகிறது
new Array() புதிய வரிசையை உருவாக்குகிறது
at() வரிசையின் குறியீட்டு உறுப்பைத் தருகிறது
concat() வரிசைகளை இணைத்து, இணைக்கப்பட்ட வரிசைகளுடன் ஒரு வரிசையைத் தருகிறது
constructor Array ப்ரோட்டோடைப் உருவாக்கிய செயல்பாட்டைத் தருகிறது
copyWithin() வரிசை உறுப்புகளை வரிசைக்குள், குறிப்பிடப்பட்ட நிலைகளுக்கு நகலெடுக்கிறது
entries() விசை/மதிப்பு ஜோடி வரிசை மறு செய்கை பொருளைத் தருகிறது
every() ஒரு வரிசையில் உள்ள ஒவ்வொரு உறுப்பும் ஒரு சோதனையில் தேர்ச்சி பெறுகிறதா என சரிபார்க்கிறது
fill() வரிசையில் உள்ள உறுப்புகளை நிலையான மதிப்புடன் நிரப்புகிறது
filter() ஒரு சோதனையில் தேர்ச்சி பெறும் ஒவ்வொரு உறுப்புடன் புதிய வரிசையை உருவாக்குகிறது
find() ஒரு சோதனையில் தேர்ச்சி பெறும் வரிசையில் உள்ள முதல் உறுப்பின் மதிப்பைத் தருகிறது
findIndex() ஒரு சோதனையில் தேர்ச்சி பெறும் வரிசையில் உள்ள முதல் உறுப்பின் குறியீட்டு எண்ணைத் தருகிறது
findLast() ஒரு சோதனையில் தேர்ச்சி பெறும் வரிசையில் உள்ள கடைசி உறுப்பின் மதிப்பைத் தருகிறது
findLastIndex() ஒரு சோதனையில் தேர்ச்சி பெறும் வரிசையில் உள்ள கடைசி உறுப்பின் குறியீட்டு எண்ணைத் தருகிறது
flat() துணை-வரிசை உறுப்புகளை இணைக்கிறது
flatMap() அனைத்து வரிசை உறுப்புகளையும் மேப்பிங் செய்து புதிய தட்டையான வரிசையை உருவாக்குகிறது
forEach() ஒவ்வொரு வரிசை உறுப்புக்கும் ஒரு செயல்பாட்டை அழைக்கிறது
from() ஒரு பொருளிலிருந்து வரிசையை உருவாக்குகிறது
includes() ஒரு வரிசை குறிப்பிடப்பட்ட உறுப்பைக் கொண்டுள்ளதா என சரிபார்க்கிறது
indexOf() வரிசையில் ஒரு உறுப்பைத் தேடி அதன் நிலையைத் தருகிறது
isArray() ஒரு பொருள் வரிசையா என சரிபார்க்கிறது
join() வரிசையின் அனைத்து உறுப்புகளையும் ஒரு சரத்தில் இணைக்கிறது
keys() அசல் வரிசையின் விசைகளைக் கொண்ட வரிசை மறு செய்கை பொருளைத் தருகிறது
lastIndexOf() வரிசையில் ஒரு உறுப்பைத் தேடுகிறது, முடிவிலிருந்து தொடங்கி, அதன் நிலையைத் தருகிறது
length வரிசையில் உள்ள உறுப்புகளின் எண்ணிக்கையை அமைக்கிறது அல்லது தருகிறது
map() ஒவ்வொரு வரிசை உறுப்புக்கும் ஒரு செயல்பாட்டை அழைப்பதன் விளைவாக புதிய வரிசையை உருவாக்குகிறது
of() வாதங்களின் எண்ணிக்கையிலிருந்து வரிசையை உருவாக்குகிறது
pop() வரிசையின் கடைசி உறுப்பை அகற்றி, அந்த உறுப்பைத் தருகிறது
prototype ஒரு Array பொருளுக்கு பண்புகள் மற்றும் முறைகளைச் சேர்க்க அனுமதிக்கிறது
push() வரிசையின் முடிவில் புதிய உறுப்புகளைச் சேர்க்கிறது, மேலும் புதிய நீளத்தைத் தருகிறது
reduce() வரிசையின் மதிப்புகளை ஒரு தனி மதிப்பாகக் குறைக்கிறது (இடமிருந்து-வலமாக செல்லும்)
reduceRight() வரிசையின் மதிப்புகளை ஒரு தனி மதிப்பாகக் குறைக்கிறது (வலமிருந்து-இடமாக செல்லும்)
reverse() வரிசையில் உள்ள உறுப்புகளின் வரிசையை மாற்றுகிறது
shift() வரிசையின் முதல் உறுப்பை அகற்றி, அந்த உறுப்பைத் தருகிறது
slice() வரிசையின் ஒரு பகுதியைத் தேர்ந்தெடுத்து, புதிய வரிசையைத் தருகிறது
some() வரிசையில் உள்ள எந்த உறுப்புகள் ஒரு சோதனையில் தேர்ச்சி பெறுகின்றனவா என சரிபார்க்கிறது
sort() வரிசையின் உறுப்புகளை வரிசைப்படுத்துகிறது
splice() வரிசை உறுப்புகளைச் சேர்க்கிறது அல்லது அகற்றுகிறது
toReversed() வரிசை உறுப்புகளின் வரிசையை மாற்றுகிறது (புதிய வரிசைக்கு)
toSorted() வரிசையின் உறுப்புகளை வரிசைப்படுத்துகிறது (புதிய வரிசைக்கு)
toSpliced() வரிசை உறுப்புகளைச் சேர்க்கிறது அல்லது அகற்றுகிறது (புதிய வரிசைக்கு)
toString() வரிசையை சரமாக மாற்றுகிறது, மேலும் முடிவைத் தருகிறது
unshift() வரிசையின் தொடக்கத்தில் புதிய உறுப்புகளைச் சேர்க்கிறது, மேலும் புதிய நீளத்தைத் தருகிறது
valueOf() வரிசையின் அடிப்படை மதிப்பைத் தருகிறது
with() புதுப்பிக்கப்பட்ட உறுப்புகளுடன் புதிய வரிசையைத் தருகிறது
